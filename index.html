Unix Command 指令

於視窗下方工具列搜尋框輸入 git bash

  • pwd: 顯示目前工作的位置

  • ls: ls ( list all files inside directory ) 顯示目前資料夾內的檔案目錄

  • cd folderName: cd ( change directory ) 進到指定資料夾

  • mkdir folderName: mkdir ( make directory ) 新增資料夾

  • touch fileName:　新增檔案，ex. touch index.html

  • cd ..: 回到上層

  • rm fileName: rm ( remove file ) 移除檔案

  • rmdir folderName: 移除資料夾，但必須是 空的資料夾

  • rm -rf folderName: 強制刪除資料夾

  • 鍵盤ctrl + L : 可讓頁面更乾淨，但不會消失code

  • 鍵盤 ↑ ↓ : 可讓頁面更乾淨，但不會消失code

  • code folderName: 開啟 VS code 打開指定資料夾(在指定資料夾的上層打開)

  • 鍵盤tab: 利用鍵盤tab去尋找關鍵字
